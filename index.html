Как запустить по ссылке

  1. Создать Google Таблицу и вставить Apps Script из файла apps_script.gs.
  2. Опубликовать Apps Script как Web App.
  3. Вставить URL в переменную API_URL в index.html.
  4. Залить index.html на GitHub Pages.
  5. Менеджеры открывают одну ссылку, нажимают “Обновить базу из облака”, вводят SKU и получают слинковку.
